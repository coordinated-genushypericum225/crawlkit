# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in CrawlKit, please report it responsibly:

1. **Email:** hieu766886@gmail.com
2. **Subject:** `[SECURITY] CrawlKit — <brief description>`

Please do **not** open a public GitHub issue for security vulnerabilities.

We will respond within 48 hours and work with you to resolve the issue.
