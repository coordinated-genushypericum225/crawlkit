SDK Quick Start

Install the SDK and start scraping with your API key:

🐍 Python

PyPI →
Install:
pip install crawlkit
Usage: 
from crawlkit import CrawlKit

ck = CrawlKit("your-api-key")
result = ck.scrape("https://example.com")
print(result.content)

📦 Node.js

npm →
Install:
npm install paparusi-crawlkit
Usage: 
const { CrawlKit } = require('paparusi-crawlkit');

const ck = new CrawlKit('your-api-key');
const result = await ck.scrape('https://example.com');
console.log(result.content);

🔧 REST API (cURL)

Usage: 
curl -X POST https://api.crawlkit.org/v1/crawl \
  -H "X-API-Key: your-api-key" \
  -H "Content-Type: application/json" \
  -d '{"url": "https://example.com"}'

API Documentation

Authentication

Include your API key in the X-API-Key header:

curl -H "X-API-Key: your_api_key_here" \
https://api.crawlkit.org/v1/crawl

Crawl Endpoint

POST /v1/crawl

Extract content from any URL.

{
"url": "https://vnexpress.net/article",
"format": "markdown" // optional: markdown | html | json
}

Video Transcript

POST /v1/video/transcript

Get video transcripts from YouTube, TikTok, etc.

{
"url": "https://youtube.com/watch?v=..."
}

Batch Processing

POST /v1/batch

Process multiple URLs in parallel.

{
"urls": [
"https://example.com/1",
"https://example.com/2"
]
}
